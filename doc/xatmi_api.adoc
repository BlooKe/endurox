== Poney Races

Poney races can be read, created, updated and deleted by a user having the 
RACE_ORGANIZER role. A poneydrome ID is used to identify where to read 
or save the poney race.

=== Read

| URL
| /api/admin/[poneydromeId]/races/[raceId]
